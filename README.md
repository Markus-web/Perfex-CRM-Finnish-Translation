# Perfex CRM Finnish Translation

Finnish (Suomi) language pack for [Perfex CRM](https://www.perfexcrm.com/).

## Installation

1. Download or clone this repository
2. Copy the files to `/application/language/finnish/` in your Perfex CRM installation
3. Go to **Setup > Settings > Localization** and select **Finnish**
